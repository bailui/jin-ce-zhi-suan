# 加密货币版使用说明

当前项目已经调整为默认面向加密货币回测，默认配置如下：

- 资产类别：`crypto`
- 数据源：`binance`
- 默认标的：`BTCUSDT`、`ETHUSDT`、`SOLUSDT`
- 默认策略：`10 - Crypto EMA 趋势跟随`
- 交易规则：`7x24`、`T+0`

## 1. 启动

```bash
cd "/Users/bailu/Documents/New project/jin-ce-zhi-suan"
./start_local.sh
```

浏览器打开：

```text
http://127.0.0.1:8000
```

## 2. 页面如何用

建议直接在输入框手动输入交易对：

- `BTCUSDT`
- `ETHUSDT`
- `SOLUSDT`

然后：

1. 数据源选 `Binance`
2. 策略选 `10 Crypto EMA 趋势跟随`
3. 选择回测区间
4. 点击“立即回测”

## 3. 命令行回测

你也可以直接命令行运行：

```bash
source .venv/bin/activate
python run_backtest.py --stock BTCUSDT --start 2024-10-01 --end 2025-04-10 --capital 100000 --strategy 10
```
